# Workspace
Just a playground for me to test code in

## Codeblocks

### Build Options
click: Project -> Build options

#### Compiler settings
check: Have g++ follow the C++11 ISO C++ language standard

#### Linker settings
Add: -l(name of library)

#### Search directories

##### Compiler
Add: /(path to include folder)

##### Linker
Add: /(path to lib folder)
